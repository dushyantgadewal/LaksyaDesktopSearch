Lakshya Desktop:

Lakshya  is a derived from Indian Language (Hindi), Which means  ' Goal ' or 'Final Target'. Lakshya Desktop is designed to search file content more easily. Since this is Open Source Project one can download, modify and redistribute the application, with restriction not to omit the Original Name of Application, Author Name (Files)  and Sponsor's (providing support of this project).

This software is design keeping in mind that the basic knowledge is necessary to build any kind of software present today. Hence the software (Source code ) provide the basic details like Working of Web Server , Indexing Technique , Reading and Writing Files , Use of related Open source projects  and many more. One can gather enough information for there own project, or help in modify this software by providing there valuable Knowledge.
